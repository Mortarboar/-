# 114學測自然科級分快速計算器 (GSAT Science Score Calculator)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Web%20%7C%20Android-green.svg)

這是一個專為大學學科能力測驗(學測)自然科設計的分數與級分轉換工具。透過簡單的點擊扣分機制,考生可以快速計算自己模考或歷屆的得分並以114學測的級距推知自己的級分。

## 功能特色

- **多選題優化**：提供專用的 `-0.8` 與 `-1.6` 扣分按鈕，方便處理多選題錯一選項或錯兩選項的情況。
- **即時換算**：輸入扣分後，系統自動根據 114 學測級距表即時更新原始分數與對應級分。
- **返回上一步 (Undo)**：支援撤銷功能，若點選錯誤可快速恢復，不需重頭開始。
- **離線可用**：採純 HTML/JS 開發，不需連網即可在手機 APK 或網頁上運作。

## 數據來源

本工具之級距計算基準參考 **114學年度學科能力測驗原得總分與級分對照表**：

| 科目 | 級距 (L) | 15級分底限 (14L) |
| :--- | :--- | :--- |
| **自然** | 7.87467 | 110.25 |

> *註：詳細級距區間已內建於程式碼邏輯中。*

## 如何使用

### 網頁版
1. 直接下載 `index.html`。
2. 使用瀏覽器（Chrome, Safari, Edge）開啟即可使用。

### Android 版 (APK)
1. 本專案可透過 WebIntoApp 等工具轉換為 APK 安裝檔。
2. 安裝後即可在手機桌面作為獨立 App 使用。
